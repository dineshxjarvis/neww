# Premium React Authentication System

A modern full-stack authentication system powered by **React**, **Vite**, **Express**, and a hybrid cloud database architecture.

## 🚀 How to Run Locally

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server** (Frontend):
    ```bash
    npm run dev
    ```

3.  **Start Backend Server**:
    ```bash
    npm run server
    ```

4.  **Access the App**:
    Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🛠 Tech Stack

-   **Frontend**: React 18, Vite, Framer Motion, Axios, Bootstrap 5.
-   **Backend**: Node.js, Express.js.
-   **Databases**: 
    -   **MySQL (Aiven)**: Core user credentials.
    -   **MongoDB (Atlas)**: Extended user profiles.
    -   **Redis (Upstash)**: Secure session management.

## 🔐 Features

-   **React Routing**: Seamless page transitions without reloading.
-   **Animated UX**: Smooth transitions powered by Framer Motion.
-   **Modern Validation**: Real-time password strength checks in React state.
-   **Secure Sessions**: Redis-based authentication.
